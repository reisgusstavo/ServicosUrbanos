# Sistema de Serviços Urbanos

## Tecnologias
- Python com Flask
- SQL Server
- HTML e CSS

## Como rodar o projeto

1. Clone o repositório
2. Instale as dependências:
   pip install flask pyodbc werkzeug

3. Execute o banco.sql no SQL Server Management Studio

4. Configure a conexão no app.py:
   SERVER=seu_servidor

5. Rode o sistema:
   python app.py

6. Acesse no navegador:
   http://localhost:5000
